Student Performance Predictor

Student Performance Predictor

Machine Learning using Linear Regression

Using Student Performance data from UCI Machine Learning repository I built a Linear Regression model to predict student final exam scores based on certain data about that student such as scores in previous tests and absence days.

Step 1: I downloaded the data file, chose which features would likely be best indicators of a students success and then I had to format the data.
Step 2: I split the data into training and test data sets. I chose to seperate out 10% for the test data.
Step 3: Next I trained the model using scikit learn to find a "best-fit" line from the student data provided. I then compared the models predicted score with the actual score from the test set data to find an accurancy percentage. Based on this I played around with which student information I included in the model in order to increase the model accuracy score.
Step 4: Finally I create a small user interface where a user can type in details such as previous test scores and absenses of a student to get a prediction of their final year score.

Sudoku Solver

Sudoku Solver

Backtracking Algorithm

The number of potential solutions to try in a given Sudoku puzzle is huge, using a backtracking algorithm enables a computer to solve a puzzle extremely quickly and efficiently.

The code works by moving along the empty squares and provisionally allocating the first number that works given then numbers in the puzzle at that point in time. It then moves on to the next empty square. As soon as it is realised that there is a square in which no number meets the requirements the algorithm moves back to the previously given provisional number and increases the provisional number to see if that satisfies the requirement. This continues until the puzzle is solved.

To confirm that value press the ENTER key on that box. To delete a pencil in you can click DEL. Finally to solve the board press SPACE, sit back and watch the algorithm run.
